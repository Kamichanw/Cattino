TaskGraph#

class cattino.tasks.TaskGraph[source]#

TaskGraph is used to specify the dependencies between tasks. Although it allows task groups to be added directly, each node within it still represents a single task.

__init__()[source]#
add_edge(u: AbstractTask, v: AbstractTask)[source]#

Add a dependency edge from task u to task v, which means task u must be executed before task v. If u or v is a task group, all tasks from u will add edges to all tasks in v.

add_edges_from(edges: List[Tuple[AbstractTask, AbstractTask]])[source]#

Add multiple dependency edges to the task graph.

add_task(task: AbstractTask)[source]#

Add a task or task group to the task graph.

add_tasks_from(tasks: Sequence[AbstractTask])[source]#

Add multiple tasks or task groups to the task graph.

filter_tasks(predicate: Callable[[Task], bool]) → DiGraph[source]#

Filter tasks in the graph based on a predicate function. The predicate function should take a task as input and return True or False.

Parameters:

predicate (Callable) – A function taking a task as input, which returns True if the task should appear in the subgraph.

Returns:

A read-only graph view of the task graph, filtered by the predicate.

Return type:

nx.DiGraph
